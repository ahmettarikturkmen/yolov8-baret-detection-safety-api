# yolov8-baret-detection-safety-api
Gelişmiş YOLOv8 modeli kullanılarak tasarlanmış, baret (hard hat) tespitine odaklanan tam kapsamlı bir güvenlik API'sıdır. FastAPI aracılığıyla görüntü ve video işleyebilir, tespit sonuçlarını Hugging Face LLM kullanarak resmi, doğal dil raporlarına dönüştürebilir.
